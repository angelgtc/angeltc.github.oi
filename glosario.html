Glosario

Algoritmo
Conjunto de instrucciones paso a paso que una computadora sigue para resolver un problema o realizar una tarea.
Aprendizaje automático (Machine Learning)
Subcampo de la IA que permite a las máquinas aprender de datos sin ser programadas explícitamente para cada tarea.
Big Data
Big data se refiere a conjuntos de datos extremadamente grandes y complejos que no pueden gestionarse ni analizarse fácilmente con las herramientas tradicionales de procesamiento de datos, en particular las hojas de cálculo. Los big data incluyen datos estructurados (Base de datos), datos no estructurados (publicaciones sociales) y conjuntos de datos mixtos (llenguaje para IA)
Inteligencia Artificial (IA)
Disciplina que busca desarrollar sistemas capaces de realizar tareas que normalmente requieren inteligencia humana
Informática.
Es la ciencia que se dedica al estudio, diseño, desarrollo y aplicación de sistemas computacionales que permitan procesar información digital de manera automática y en diversos formatos.
Chatbot
Programa que simula una conversación con usuarios humanos, usando IA para comprender y responder.
Interconectividad
En informática, se refiere a la capacidad de sistemas, redes o dispositivos de conectarse y comunicarse entre sí, incluso si son heterogéneos, para intercambiar datos o realizar tareas de manera conjunta
Clasificación
Tarea de IA en la que se asignan categorías a datos según sus características.
Datos de entrenamiento
Información utilizada para enseñar a un modelo de IA cómo realizar una tarea.
Etica de la IA
Rama que analiza las implicaciones morales del uso y desarrollo de la inteligencia artificial. -Generalización: Capacidad de un modelo de IA para aplicar lo aprendido a nuevos datos no vistos.
IA débil
También llamada IA estrecha. Está diseñada para realizar una tarea específica (por ejemplo, Siri o Alexa).
IA fuerte
Hipotética IA que puede razonar y tomar decisiones como un ser humano.
